# Wazuh SOC Lab

A home SOC lab built with Wazuh SIEM and Docker, demonstrating real-world blue team
detection and response scenarios.

## Environment

- **SIEM:** Wazuh (Docker deployment)
- **Agent:** Ubuntu Linux VM
- **Host:** Ubuntu LTS 24.04
